Attributes and Links

Attributes

Attributes are used to modify elements. Attributes are used in a number of ways and are useful when adding links, images, accessibility, adding style with CSS and interactions with Javascript. Today, we are going to use them to create links.

Links

Links are created by using the anchor element. There are many different kinds of links you can make. For example, you can link to an email address, an image, or some other service.
