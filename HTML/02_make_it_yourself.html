Your first webpage

The index page

Websites have a special file for the first page you come to on a site. When you visit google.com or yahoo.com, or any domain at all, you're actually accessing a file named index.html.

We'll make this index page in a few minutes, but first let's look at the structure of HTML, the markup language we'll be using.

HTML structure

HTML defines the structure of your webpages. The language contains elements that are typically comprised of two tags. Each HTML element tells the browser soemthing abotu the information that is inside two tags. As we mentioned earlier, it's useful to think of HTML as the bones of your site. Just like you, HTML contains a head and a body.

Tags are like containers for content. They surround the things they describe.

Below is text for a sample HTML page.

example HTML code

<html>
    <head>
        <title>Hello World</title>    
    </head>  
    <body>   
        <p>This is my site!</p>
    </body>   
</html>

What all that stuff means

The text above is a very simple webpage. You can see what the page looks like when rendered

The elements inside the less-than and greater-than symbols are HTML tags. In this page, you can see the html, head, title, body, and p tags. Let's address these individually.

  • html - this element goes at the beginning of every HTML page. it is closed with /Hhtml when you're finished with the file
  • head - This part of the file will contain information like the site's title, and any additional files you want to use, like CSS or javascript.
  • title - This is the title of your webpage. it, and it's closing tag, are between the head and the /head tags
  • body - The body contains the content you will read off the scree. it;s where you will put the majority of your content, text, audio, video, images, links, and so on.
  • p - This is a paragrapgh tag. Text will go between the two p tags and will be displayed in the main browser window.
