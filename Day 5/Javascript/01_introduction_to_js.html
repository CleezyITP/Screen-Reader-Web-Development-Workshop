Javascript basics

What is Javascript?

HTM and CSS are called markup languages. They describe certain elemetns of a document. Javascript is a programming language. It allows you to change elements on the page based on user actions. Just like HTML and CSS, you can write javascript in notepad, and it will run in . your browser.

The language

Javascript has some new qualities that are part of most programming languages. We'll discuss two of them, variables and functions.

Variables

A variable is something you define in your program that can store a number or word that can change over time. Think of it like a backpack. You might have a computer in a backpack, or a shirt, or some food. you can change the contents whenever you like. variables are used by programmers to store and change data over time.

You write a vaiable in javascript like this

var myVariableName = somevalue ;

Functions

Functions are blocks of code that do one thing. You invoke a function by calling its name when an event happens. Web evenets might be selecting a button, loading a page, or entering text into a field. When you write a fucntion, it becomes a collection of code that you can reuse over and over again.

You write a function in javascript like this


function myFunction() {
}

Your code to execute a change in the page would go inside the braces.
